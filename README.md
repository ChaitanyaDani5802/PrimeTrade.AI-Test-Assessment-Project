## PrimeTrade.AI-Test-Assessment-Project

# Crypto Sentiment vs. Trader Performance Analysis

This project explores the intersection of **on-chain execution data** (from Hyperliquid) and **market psychology** (Bitcoin Fear & Greed Index). The goal is to determine if market sentiment is a reliable leading indicator for retail trader profitability.

## 💡 Key Objectives
1. **Correlation Mapping:** Do traders lose more money during "Extreme Greed"?
2. **Behavioral Analysis:** Does trade size (volume) increase as fear rises?
3. **Strategy Validation:** Identify the "Sweet Spot" sentiment for high-probability entries.

## 📂 Dataset Overview
- `historical_data.csv`: Real execution logs including `Closed PnL`, `Side`, `Size`, and `Execution Price`.
- `sentiment_data.csv`: Historical daily classification of Bitcoin market sentiment (Fear vs. Greed).
